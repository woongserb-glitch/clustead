# 브랜드 공식 로고 (자동 감지)

Home 그래프의 3차 브랜드 노드는 기본적으로 **브랜드 컬러 워드마크 칩**(흰 배경 + 브랜드색 텍스트)으로 표시됩니다.

공식 로고를 표시하려면 **이 폴더에 `<슬러그>.svg`(또는 `.png`/`.webp`) 파일만 넣으면 됩니다.**
서버가 폴더를 스캔해 자동으로 해당 브랜드 노드에 로고를 렌더합니다. 코드 수정 불필요.

> 로고는 상표입니다. 각 브랜드의 사용 가이드라인/라이선스를 확인한 자산만 넣으세요.
> 저장소에는 트레이드마크 아트워크를 포함하지 않습니다.
